# 虫スパ

虫の力を組み合わせてビルドを作る、デッキ構築型ローグライクのプロトタイプです。

## コンセプト

- 虫は戦闘ユニットではなく、チームHP・属性補正・特殊能力を提供するビルドパーツです。
- カードが主な攻撃手段で、虫と改造パーツによってカード効果を伸ばします。
- 成長は数値インフレではなく、毒・切断・反撃・回復などのシナジーを重ねる設計です。
- 虫の強さは持ち越さず、図鑑の発見だけが永続要素になります。

## プロトタイプの内容

このリポジトリには、仕様確認用の静的Webプロトタイプを収録しています。

- 戦闘画面: チームHP、虫編成、敵の予告行動、手札、エネルギー、ログを表示します。
- カード使用: 「きる」「さす」「断頭」「防御」「つく」をクリックして使用できます。
- 虫シナジー: ノコギリクワガタ、スズメバチ、カブトムシの補正と特性がカードに反映されます。
- 報酬選択: 戦闘後に新カード・新虫・虫改造・回復を選べます。
- 虫改造: 改造パーツを装着し、チーム効果に追加できます。
- 図鑑: 発見済みの虫が登録されます。

## 起動方法

依存関係はありません。任意の静的ファイルサーバーで配信してください。

```bash
python3 -m http.server 8000
```

ブラウザで <http://localhost:8000> を開きます。
